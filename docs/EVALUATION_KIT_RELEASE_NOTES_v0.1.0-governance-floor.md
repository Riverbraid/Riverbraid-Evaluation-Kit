# Riverbraid Evaluation Kit v0.1.0 governance floor

This document provides the release body for the public Evaluation Kit starter release.

It is a visibility and release notes artifact only. It does not create the GitHub Release by itself.

## Release body

This release publishes the Riverbraid Evaluation Kit as the public starter kit for independently reproducing Riverbraid's pinned governance floor.

## What this release provides

- A 30 repository verified registry with pinned commit references
- A canonical GitHub Actions verification path on ubuntu-latest
- A Docker based runtime verification surface
- Expected output comparison through expected-results.json
- Required file presence checks across the starter kit
- Reproduction helpers for Windows and Unix environments
- Claim boundary documentation explaining what a pass does and does not mean

## Evidence boundary

This release is a public starting floor for inspection and reproduction.

It is not a certification product, legal approval mechanism, security guarantee, production deployment package, or third party audit.
