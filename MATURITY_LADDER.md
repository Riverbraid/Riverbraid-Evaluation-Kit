# Maturity Ladder

| Level | Description | Status |
|---|---|---|
| M0 | Conceptual documentation | Complete |
| M1 | Local verification | Complete where recorded |
| M2 | Remote CI verification | Current target baseline for verified registry claims |
| M3 | Independent reproduction | Target of this kit |
| M4 | Multi environment reproduction | Not yet |
| M5 | External audit | Not yet |
| M6 | Production profile certification support | Not yet |

Riverbraid should be described as between M2 and M3 until an independent evaluator successfully reproduces the kit output. This kit is the gate for M3. It is not itself proof of M3 until independently run.
